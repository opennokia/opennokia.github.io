About the OpenNokia Project

The OpenNokia project is a complete modern redesign of the iconic Nokia 3310, that is fully open source and using only off-the-shelf parts. We're combining the nostalgic design with semi-modern features, like Bluetooth, WiFi, and much more!

Features

  • Fully open source, including KiCad files etc.
  • Easily availible parts
  • Small impact on the stock case and none on the factory components
  • Includes many factory features including calling and messaging
  • New tools and apps
  • Bluetooth, WiFi and GPRS connectivity
  • And much more!

Specifications

  • MCU: LOLIN32 LITE USB-C, 520kB SRAM, 4MB Flash, builtin Li-Po charge controller
  • Wi-Fi: 802.11 b/g/n
  • Bluetooth: v4.2 BR/EDR, BLE
  • GSM/GPRS: SIM800L
  • Battery: Unknown, user choice
  • Factory case, screen, keypad, speakers, microphone, vibration motor
